# Siirtoäänivaali 2022

TKO-äly ry:n vuoden 2022 [äänestys- ja vaalijärjestyksen](https://www.tko-aly.fi/attachments/files/324/2022-12-28-_nestys-ja-vaalij_rjestys.pdf?1672444809) mukainen ääntenlaskuohjelma.
